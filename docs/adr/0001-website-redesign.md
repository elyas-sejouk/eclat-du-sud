# 0001 Website Redesign from Playful to Professional

We have decided to completely redesign the website design language for "Éclat de Sud", transitioning from a playful, neo-brutalist aesthetic (thick black borders, heavy flat shadows, vibrant yellow/green color blocks, and emojis) to a premium, professional corporate aesthetic. The new design system uses a deep navy and sapphire blue primary color palette with warm gold accents, elegant typography (Outfit for headings, Inter for body), thin clean slate borders with soft shadows, and clean SVG line iconography. This choice aligns the website visual identity with the expectations of high-end business clients, particularly property managers (Airbnb turnarounds) and construction companies.
